2019 US Hate Crime Data Visualization Across the Map

Each graph focuses on different cateogrizations of hate crimes (categories of motivation include: Ethnicity/Race, Religion, and Sexual Orientation).

Datasets used to create visualizations come from the FBI Uniform Crime Report public dataset. The information was scraped using Python into a format usable by Tableau to create these maps

Note: This data is for 47 contiguous states (Excludes Alaska and Hawaii for formatting reasons. Alabama was not reported under the FBI UCR.
